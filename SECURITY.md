# Security Policy

## Reporting a Vulnerability

Please report security issues privately via GitHub Security Advisories:
<https://github.com/alaskacg/alaska-status/security/advisories/new>

Alternatively: security@alaskaconsultinggroup.com

We aim to respond within **72 hours** and patch critical issues within **7 days**.

## Scope

This repository is part of the Alaska Consulting Group portfolio (monitored by
Trinity/Morpheus). Security updates from Dependabot are auto-merged when CI passes
and the update is a patch or non-breaking minor.

## Out of Scope

- Denial of service via free-tier rate limits
- Findings already reported by Dependabot/CodeQL
- Issues in third-party services we don't operate
